assume NO responsibility.

**IF THIS SCREWS UP YOUR FINANCES, DON'T BLAME ME!**   

## To Build

To checkout Flunky and compile it, execute the following commands:

    cd <workspace-dir>
    git clone git@github.com:lynchnf/dough.git
    cd dough
    git checkout main
    mvn clean package
